# Google Search Console Submission Steps

After the site is deployed at `clearout.aurorasitesolutions.com`, do this:

1. Open Google Search Console.
2. Add property: `https://clearout.aurorasitesolutions.com`.
3. Verify ownership using the DNS TXT record or HTML tag method shown by Google.
4. Open **Sitemaps**.
5. Submit this sitemap path:

```text
sitemap.xml
```

Full sitemap URL:

```text
https://clearout.aurorasitesolutions.com/sitemap.xml
```

6. Use URL Inspection on the homepage and several important pages:

```text
https://clearout.aurorasitesolutions.com/
https://clearout.aurorasitesolutions.com/furniture-removal-calgary
https://clearout.aurorasitesolutions.com/garage-cleanout-calgary
https://clearout.aurorasitesolutions.com/move-out-cleanout-calgary
https://clearout.aurorasitesolutions.com/junk-removal-beltline-calgary
https://clearout.aurorasitesolutions.com/junk-removal-panorama-hills-calgary
https://clearout.aurorasitesolutions.com/junk-removal-mahogany-calgary
```

7. Request indexing for the homepage and the highest-priority service/community pages.

## What not to do

Do not submit old short community URLs as canonical sitemap URLs. The canonical versions are:

```text
/junk-removal-[community]-calgary
```
