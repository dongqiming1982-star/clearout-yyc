# V22 Clean SEO Package

This package removes the Supabase/Postgres marketplace foundation and Stripe webhook files from the previous v22 bundle. It is intended for the SEO launch site only: public pages, request form, provider beta form, sitemap, robots, and structured data.

Use this version if you want the simplest install/deploy path first.

Run:

```bash
npm install
npm run build
```

If npm install appears stuck, press Ctrl+C and run:

```bash
npm install --verbose
```
