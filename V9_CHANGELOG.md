# v9 Changelog

- Customer estimator simplified into a main flow plus optional details.
- Customer lead data is fully structured for future database and dispatch rules.
- Provider proof fields changed from plain filenames to upload-ready metadata.
- Provider availability changed from free text to structured days/time windows.
- Provider daily lead limit changed to structured numeric selection.
- Provider approval is not automatic; local demo saves applications as not active.
- Supabase schema updated for structured leads, provider review, and dispatch filtering.
- API stubs updated to accept structured lead/provider payloads.
